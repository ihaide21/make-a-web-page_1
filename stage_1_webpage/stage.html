Lecture 1: Overview of the web

The web is a bunch of computers linked together that communicte with each other. When a user requests a web page through a browser (eg. Google Chrome, Safari), their computer sends a HTTP Request to a server.
In response, the server finds the relevant HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the screen.

What is HTML

HTML stands for Hypertext Markup Language. HTML documetns form the majority of the contents on the web. It provides a basic structure to a webpage, just as walls provide framework for rooms and hallways.
It is written in the form of HTML elements anc web browsers can read HTML files and compose them into visible web pages that are easy to navigate through

Tags and Elements

HTML elements form the building blocks of all websites. Most HTML elements have a start tag and an end tags that indicate where the element begins and where it ends. Those with exception to this rule are called empty/void tags. These tags, such as, breaks tags, don't have closing tags since they don't have any content.

Inline vs. Block elements

Block elements form an invisible box around the content inside of them and normally start and end with a new line. Examples of block elements include div tag, h tag, and p tag...
Inline elements are displayed without line breaks. Examples include span and break tag.

Div tag

Div tag is an example of a 'block element'. Div tags are used mainly for two reasons:
1. To logically organize our code 2. They group our items together so we can add styling to the whole block at once. This avoids repetition - a key concept in CSS.

Text editors

When writing code, programmers use special text editors (Sublime Text, Text Wrangler). These editors make the programmer's life easier. For example, some text editors will automatically generate a closing HTML tag when you write an opening tag.

What is CSS

CSS stands for Cascading Sytle Sheet. It is a powerful tool that enables us to add style to HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

At first, I had to change the color of each paragraph individually, thereby, repeating the process 10 times. As soon as I realized this repetition, the concept of class was easy to understand then.
